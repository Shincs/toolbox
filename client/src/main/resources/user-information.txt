USER INFORMATION - PLEASE READ BEFORE USE
1. Der Spicker öffnet sich im Systemtray (Windows) oder im Dock (Mac OS X).
2. Panikknopf = Rechtsklick auf die Anwendung.
3. hehehea.
4. Der Spicker kann über den Systemtray wieder geöffnet werden.
5. Der Spicker kann über den Systemtray beendet werden.
6. Über das Systemtray Icon kann ein silent Screenshot erstellt werden.
   -> Dieser wird im Appdata Verzeichnis gespeichert.
7. Bilder können in den Image Switcher gezogen und mit den Pfeiltasten bedient werden.
8. Google ist integriert.
9. Settings werden gespeichert.
10. Es gibt einen Chat.
